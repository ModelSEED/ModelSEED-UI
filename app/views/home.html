About


ModelSEED is a resource for the reconstruction, exploration, comparison, and analysis of metabolic models.

  • ✓ Faster reconstruction and gapfilling of models from annotated genomes.
  • ✓ Integrated interface for FBA and gapfilling.
  • - Gapfill on your own custom media, run sophisticated FBA simulations, etc.
  • ✓ Greatly improved application performance.
  • ✓ API for programmatic access to modeling tools and data.

Utilized Data Sources

Core Models were designed based on the information derived from published literature. Genomes were annotated using RAST annotation pipeline and core models were generated using KBase resources. We've done our best to cite contributing data sources here. If you believe your data has been utilized and not properly cited, please email us here.
