Main features

1. The Excel-like subsystem table consists of columns that have the functional roles as captions, and rows that have genome names as captions. As in Excel, the columns can be scrolled from left to right, and the rows from top to bottom, while the column and row headers stay visible.

2. The first row of the table lists the reactions associated with the corresponding functional role of each column. The reaction ids are clickable and can open a separate webpage that will display more biochemistry information on that reaction.

3. Below the reaction row, each table cell consists of three groups of gene annotations: curation, candidates and predications.
Genes in curation are colored green, in predications are colored blue.

4. You can select gene id entries from one group and assign it to another, as the arrows point. If a gene in candidates is not included in either curation or predications, that gene id is shown as red; otherwise it will be black.

5. By hovering your mouse cursor over the candidates dropdown box, you will see a prompt indicating that you can double click on any of the entries. A popup box will be displayed to allow you to make changes. After you exit the popup window, you will be reminded to save or save as your changes.

6. By hovering your mouse cursor over any of the column caption headers, you will see a prompt indicating that you can right click on the column caption to reveal the protein family/families associated with this function role. You can then select one of the family name to open a family tree window. You will be presented with the selected protein family tree that has mapped the annotations in the cells of the selected functional role column with the tree leaf nodes.

7. If a match is found, the corresponding colors (green=curation, blue=predications, red=neither).
If the leaf node is not matched at all, then a blank spot by that leaf node is reserved. In addition, the confidence score out of 100 is also presented.

8. Please feel free to zoom in and out the tree to view more details. By clicking on the tree nodes, specific node data will also be presented.
